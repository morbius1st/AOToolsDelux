Delux Measure Help Information


This document describes Delux Measure and the Unit Style apps

Introduction

I am sure that you find the standard measure tool a bit lacking as the measured distance is only the direct XY distance. I know that frequently, I want separate X & Y distance and using the standard measure tool means either measuring twice or placing a temporary dimension - not the most efficient system.

Which is why I created, Delux Measure.
 
With Delux Measure, you can pick two points and get the full range of geometric distances between the points (within the limitations imposed by Revitï¿½ - more on this below).  This includes the X, Y, and Z distances as well as the 4 diagonal distances XY, XZ,YZ, and, of course XYZ.  These distances are presented using a unit style of your choice (and design - more on this below) as well as an secondary unit style.  I find this most helpful when working on site plans since the Architectural site plans will, typically, use a Feet and Fractional unit style but the Civil Engineer's plans use Decimal Feet.  The conversion math is simple but tedious having to convert each time.  Having two unit systems at the same time greatly simplifies this task.

Interface

Once installed and loaded, Delux Measure creates a single split ribbon button with Unit Style shortcuts.  In addition, there is a drop down with two buttons - the Delux Measure command and the Unit Styles Manager.  At first, having the Delux Measure commend placed on the drop down, seems to not make sense.  But I have a reason for this ...  To get the most out of Delux Measure, I think you will want to access it quickly.  To do this, my expectation is that you create a keyboard shortcut.  

Create a keyboard shortcut, if you do not already know how,select the View tab ► User Interface Button ► Keyboard Shortcuts command (at bottom of the list).   Search for "CsDeluxMeasure".  This should find a single entry: "External Tools:CsDeluxMeasure".  Select this item and assign a shortcut - I use "DX" myself.  Once you've done and saved this, you can access Delux Measure without using a ribbon button by just typing DX in most situations - this is the most efficient method.

Unit Styles

One of the main fextures is unit-styles. A unit style is a group of distance unit settings saved and named for future reference. Unit styles are created, organized, and saved using the Unit Styles Manager.

Unit styles are used in two ways.
  • in the Ribbon Button
  • in the Delux Measure window
Each of these is described below.

Components

As noted, there are three primary parts - Delux Measure, the Ribbon Button with Unit Style shortcuts, and the Unit Styles Manager.

The Ribbon Button with Unit Style Shortcuts is described below.

Select this to get Information about Delux Measure.

Ribbon Button with Unit Styles

The ribbon button, pictured here, shows an example of what it would look like:

The Ribbon Button is a split-button with the last selected unit-style showing.  Pressing this button changes the project distance units the the values saved in the unit style. This allows you to quickly switch between distance unit settings.
Maybe you are thinking - why would I need to change the project distance unit settings. Fair question. I found this to be very helpful in these situations. I'm sure you can think of others:
  • When workiing on a family type, it makes more sense to have the project units be fractional inches or decimal inches rather than feet and fractional inches.  This happens a lot for me when working on smaller items such as balusters, or other small component.
  • Needing to switch between Imperial and Metric
  • When working on a site plan and having to croordinate with drawings from a civil engineer who uses decimal feet.
Below the Ribbon Button is the drop-down which contains the Unit Styles Manager and the Delux Measure.
